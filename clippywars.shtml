Clippy Wars
24th of September 2017
For the class Game Prototyping at ITU we were given the assignment to create a prototype which invokes an emotion in the player. Our group focussed on confusion for the prototype.

We came up with Clippy Wars. Clippy Wars is a platformer in which the player has to grab an object(the hand) and hold it for 15 seconds. So far it sounds fairly easy right, The only thing is, though, that there are four player characters which look exactly like each other. Furthermore, they all copy your movement with some random delay. Therefore, upon commencing a game you first have to find out which character is your real character and collect the objective. Try not to loose track of your character whilst doing this! Downloads for the game can be found here for Windows and here for MacOS.
