Empire of Dirt
26th of January 2015
Empire of Dirt is an C++ OpenFrameworks project that includes pathfinding & random generation.

For this project worked with Joep Schyns. The project is made for a second year programming course in Creative Technology. The project had to be written in C++ with the use of OpenFrameworks. This program generates a random world in which two teams are battling each other while looking for a flag. Features of the program are: A* search algorithm, randomly generated world, tanks influence the terrain, explored terrain & darkness. We wrote this program in Visual Studio 2012.
